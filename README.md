# 🎮 Game Video Compressor (多音轨游戏录像压缩工具)

这是一个基于 Python 和 FFmpeg 编写的自动化视频压缩脚本，专为追求极致体积/画质比的游戏玩家（如守望先锋等高帧率游戏）设计。

## ✨ 核心特性
* **无脑拖拽操作**：支持直接将视频文件或包含视频的整个文件夹拖入运行。
* **H.265 (HEVC) 高效编码**：同等画质下比传统 H.264 节省 30%-50% 的硬盘空间，经测试后大约是英伟达直出文件的1/5。
* **完美保留多音轨**：自动映射并保留游戏原声、麦克风等所有独立音轨。
* **可视化进度条**：告别枯燥的命令行，实时显示压制进度、当前帧率、耗时等详细参数。
* **自动去重保护**：智能跳过已压制过的 `_x265` 后缀文件，防止重复劳动。

## 🚀 如何使用
1. 在右侧的 **Releases** 页面下载打包好的 `.exe` 可执行文件。
2. 将你需要压缩的录像文件夹，直接拖拽到 `.exe` 图标上，喝杯茶。
## 🛠️ 适合人群
硬盘空间告急、需要归档大量游戏高光时刻、或是需要保留高画质素材用于美术动态参考的玩家与创作者。
